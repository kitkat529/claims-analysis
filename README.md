# Project Description:
## This project is a simulation of the type of analysis performed regularly by healthcare data analysts, medical billing specialists, and health informaticians. It involves a comprehensive analysis of healthcare claims data focused on understanding the relationship between claims, lines, and codes. Advanced analysis was performed to identify top billing providers, most frequent ICD10 codes, and providers handling the most complex cases. Data visualizations are also included to showcase findings

# Data Source:
## Sample of prospective claims data provided by SBM compliance and revenue cycle teams from May 2024

# How To Run the Notebook
## Execute each cell sequentially. First import necessary packages and load the data. Then perform data exploration, followed by relational and creative analysis

# Summary of Key Findings:
## 1. The dataset comprises of 388 unique claims from 9/25/23 to 5/29/24
## 2. The most common diagnostic code is J96.01: Acute respiratory failure with hypoxia
## 3. SB Internists are among the top billing providers, while Medicare is the primary payer with highest claim volume
